  4. 以后使用github的时候，直接（只有）使用ssh连接就可以实现免密

  1. 在github中创建一个仓库
  2. 仓库的名字必须是 用户名.github.io
  3. 将写好的静态页面push到这个cnagku
  4. 通过 用户名.github.io 这个地址就可以访问网站了

注意： 一个账户只能有一个这样的项目！！！

使用hexo creat blog

Share

步骤

  1. 需要安装node.js
  2. 通过npm下载hexo
  1. 上面一步完成之后，就有一个命令可以用了 hexo
  2. 创建一个新的文件夹
  3. cd进入这个文件夹
  4. 执行 hexo init
  5. 执行 hexo s 在本地可以进行预览
  6. 执行 hexo g 可以将md文件生成为html文件（目录为 当前博客目录下的 public目录）
  7. 想要创建新的博客 hexo new "文章名称"
  8. 要将博客发布到github上， 创建一个静态页面托管的项目
    将public目录下所有的内容全部推送到项目中即可
  9. 以后要写文章，只需要 hexo new hexo g 然后push public目录中的内容即可！
